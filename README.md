 * JAVA 11
 * Spring MVC
 * Oracle Database
 * Mybatis

## 주요 기능
▶ 사용자
 1. 회원가입/탈퇴, 카카오 간편 로그인
 2. 로그인, 로그아웃, 아이디 저장, 자동 로그인, 아이디/비밀번호 찾기
 3. 회원정보 수정
 4. 내 소모임, 찜한 소모임, 작성글, 작성 댓글 조회
 5. 소모임 가입, 검색, 소모임 찜
 6. 게시글 조회, 작성, 수정, 삭제
 7. 일정 조회, 생성, 수정, 삭제
 8. Q&A 조회, 작성, 수정, 삭제
 9. 소모임, 다른 사용자 신고



▶ 사용자(소모임장)
1. 소모임 생성
2. 소모임 가입 승인
3. 소모임 멤버 추방
4. 소모임장 위임
5. 소모임 삭제



▶ 관리자
1. Q&A 게시글 답변 작성
2. 회원 조회, 삭제
3. 소모임 조회, 삭제
